How it works

lingbot-map (arXiv:2604.14141) is a feed-forward Geometric Context Transformer that turns a video stream into camera poses + dense metric depth, frame by frame, with a paged KV cache — no per-scene optimization. We drive it one frame at a time on the local GPU and stream each frame's geometry to this browser.

 Browser (three.js)  ◀──WebSocket frames──  FastAPI (this host)
   point cloud                                  │
   + trajectory          {pose, depth, points}  │  LingbotEngine (RTX 4070, bf16, SDPA)
   + camera frustums                            ▼
   + live depth          unproject depth + K + pose ──►  lingbot-map.pt (4.6 GB, on E:)
  • Runs on the GPU here: the transformer (8 GB-safe: SDPA, CPU-offload, window=16, bf16).
  • Runs in the browser: rendering + accumulation of the point cloud and trajectory.
  • Real data, live: each frame is computed on demand, not replayed from a baked file.
